overflow: 부모요소보다 자식요소의 내용이 더 크거나 많을 때 어떻게 보일지를 선택하는 속성

overflow - visible

1. visible: 특정 요소가 박스를 넘어가도 그대로 보여준다. (기본값)
2. hidden: 부모 요소의 범위를 넘어가는 자식 요소의 부분은 보이지 않도록 숨긴다.
3. scroll: 항상 스크롤바를 표시한다.
4. auto: 내용이 넘치면 스크롤바를 표시한다.
